# CODEX FOLLOW-UP PROMPTS

## Prompt 1 — Scaffold only
Scaffold the full solution structure, all projects, core interfaces, base MVVM shell, navigation, dependency injection, logging, settings persistence, and build scripts. Make the solution compile before implementing advanced features.

## Prompt 2 — Cleaner engine
Implement the cleanup rules engine, temp folder scanning, recycle bin estimation/cleanup, exclusions, dry-run mode, and issue models. Add tests for rule execution and reclaimed-size calculations.

## Prompt 3 — Driver center
Implement device inventory, hardware ID collection, current driver metadata extraction, recommendation models, provider abstraction, result scoring, and a driver details UI. Do not implement internet scraping yet.

## Prompt 4 — Startup and repair
Implement startup inventory, startup impact scoring, orphaned startup detection, broken reference repair, uninstall leftovers analysis, scheduled task leftovers detection, and safe confirmation workflows.

## Prompt 5 — Reports and release
Implement HTML/text reports, export flows, action history, MSIX packaging scripts, Store listing starter assets, privacy/EULA templates, and release checklist outputs.
